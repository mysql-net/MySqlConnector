MySqlConnector

Use with ORMs

This library is compatible with popular .NET ORMs including:

  • Dapper (NuGet)
  • NReco.Data (NuGet)
  • ServiceStack.OrmLite (MyGet)
  • SimpleStack.Orm (NuGet)

For Entity Framework support, use:

  • Pomelo.EntityFrameworkCore.MySql (NuGet)
